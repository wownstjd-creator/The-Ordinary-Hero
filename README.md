## 조작법

A D 이동

Space 점프

Left Shift 대쉬

Q 감지

E 상호작용

Enter 대화

---

## 현재 구현된 주요 기능

- 플레이어 이동, 점프, 대쉬
- 스테미나 기반 행동 시스템
- 감지(Q) 시스템
- 감지로 드러나는 플랫폼/오브젝트
- OBS 관측 지점 시스템
- 고블린 일반 몬스터
- 고블린 킹 보스 프로토타입
- 보스 오라 패턴 예고 시스템
- 보스 스킬: 돌진, 슬램, 고블린 소환, 독안개
